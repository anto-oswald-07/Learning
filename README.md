## 🚀 Getting Started

### Adding a New Project

1. **Create a new directory** under `projects/` with a descriptive name (use kebab-case):
   ```bash
   mkdir projects/my-new-project
   ```

2. **Create a README.md** for your project using the template from `projects/example-project/README.md`

3. **Add your code** and any necessary files to the project directory

4. **Update this README** by adding your project to the Projects Catalog table above

### Project Structure Guidelines

Each project should include:
- `README.md` - Documentation explaining what the project does, how to run it, and what you learned
- Source code files
- Configuration files (if needed)
- Any additional documentation
