# Projects Directory

This directory contains all the ultra mini projects built during the learning process.

## Structure
Each project should have its own directory with:
- `README.md` - Project documentation
- Source code files
- Any necessary configuration files

## Adding a New Project
1. Create a new directory under `projects/` with a descriptive name
2. Copy the structure from `example-project/` as a template
3. Add your project files and documentation
4. Update the main README.md with your project information

## Naming Convention
Use kebab-case for project directory names:
- ✅ `todo-app`
- ✅ `weather-dashboard`
- ✅ `api-client`
- ❌ `Todo App`
- ❌ `weather_dashboard`
